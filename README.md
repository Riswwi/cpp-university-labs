# C++ University Labs — ZUT Informatyka

Laboratory assignments from the Informatyka programme  
at West Pomeranian University of Technology (ZUT), Szczecin (2025–2026).

## Labs

**Lab 1 — Functions & Memory**  
Function overloading, references, dynamic memory allocation with `new[]`/`delete[]`.  
Data: array of `Sensor` structs.

**Lab 2 — OOP & Dynamic Structures**  
Classes `Ship`, `Squadron`, `CommandCenter` with dynamic arrays,  
manual memory management, const methods.

**Lab 3 — Advanced OOP**  
`friend` classes, nested classes, `mutable` keyword, `const` references,  
`decltype`. Climate controller simulation with sensor benchmarking.

**Lab 4 — Namespaces & Operator Overloading**  
`Vector` and `Matrix` classes inside `DataStorage` namespace,  
utility class in `Utils` namespace. Overloaded operators: `[]`, `()`, `+`, `*`, `==`.  
Copy constructor, destructor, assignment operator.

## Tech
Language: C++  
Compiler: MSVC (Visual Studio)
